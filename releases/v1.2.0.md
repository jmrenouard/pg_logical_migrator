# Release v1.2.0

## [1.2.0] - 2026-04-12

### Features
- **migrator**: add multi-schema support allowing `all` or comma-separated schema lists
- **diagnostics**: implement deep database and table size analysis (data vs. index) with % DB footprint
- **monitoring**: add `repl-progress` command for real-time byte-level and table-count synchronization tracking
- **monitoring**: add real-time progress feedback to `wait_for_sync` method (tables remaining, % bytes, elapsed time)
- **validation**: add `--use-stats` option for near-instant row count estimation via `pg_stat_user_tables`
- **pipeline**: integrate `wait_for_sync` with progress display into automated workflows for improved reliability
- **pipeline**: add `--no-wait` option to `init-replication` to skip synchronization wait
- **tui**: overhaul interactive dashboard with real-time progress, size analysis, and full 16-step re-numbered workflow

### Bug Fixes
- **post-sync**: fix ownership reassignment to correctly handle and exclude automatic table-row types
- **monitoring**: fix table name matching in replication progress tracking for `public` schema
- **tui**: fix rendering errors (`NotRenderableError`) by standardizing Rich Panel usage

### Refactor
- **cli**: modularize monolithic script into `src/cli/` sub-modules (`commands`, `pipelines`, `helpers`)
- **schema**: refine `pre-data` and `post-data` separation into distinct logical steps

### Documentation
- **architecture**: add `DOCS/CODEBASE.md` for high-level technical reference
- **testing**: add `HOWTO.md` with detailed instructions for all test modes

### Testing
- **unit**: double test coverage for core modules including new size analysis and progress features
- **integration**: add specialized SQL scenarios for Large Objects (LOBs), unlogged tables, and 100M rows

### Docker
- **connectivity**: update default config to `localhost` and add `source_host` for reliable container-to-container replication
- Updated images available on [Docker Hub](https://hub.docker.com/r/jmrenouard/pg_logical_migrator) and [GHCR](https://github.com/jmrenouard/pg_logical_migrator/pkgs/container/pg_logical_migrator)
