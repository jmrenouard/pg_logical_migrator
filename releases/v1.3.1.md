# Release v1.3.1

## [1.3.1] - 2026-04-16

### Bug Fixes
- **tui**: fix bug in replication status display where publisher slots were incorrectly counted (incorrect variable name in loop)
- **tui**: improve async stability of the `init-replication` pipeline by capturing UI parameters (checkboxes) before entering background worker threads
- **tui**: add missing `@work` decorators to ensure proper async execution in Textual

### Features
- **config**: store `config_path` in `Config` class instance for better traceability
- **rollback**: block `setup-reverse` if forward replication subscription still exists on destination (prevents conflicting flows)
- **rollback**: add `cleanup-reverse` command to remove reverse replication objects
- **tests**: add unit test to verify `Config.config_path` storage
- **tests**: add unit test for `setup-reverse` blocking logic
- **tests**: add unit test for `cleanup-reverse` logic

### Maintenance
- **HOWTO**: update documentation to prefer `docker compose` (Docker Desktop/Compose V2) over legacy `docker-compose`
- **Makefile**: switch from `docker-compose` to `docker compose` for managing test environments
- **test-env**: reduce size of `large_data.sql` from 100M to 1M rows for faster and more stable E2E testing
