# Release v1.0.0

## Summary

Initial release of **pg_logical_migrator** — a Python-based utility to orchestrate PostgreSQL database migrations using logical replication.

## Features

- **14-step migration pipeline** with individual step control
- **Interactive TUI** (Textual) for guided migration operations
- **CLI interface** with `init-replication` and `post-migration` subcommands
- **Automated pipelines** for init replication and post-migration workflows
- **Pre-flight diagnostics**: connectivity, parameter checks, problematic object detection
- **Schema migration** via `pg_dump`/`pg_restore` with optional target drop
- **Logical replication setup**: publication, subscription, and replication slot management
- **Post-sync operations**: sequence sync, trigger management, materialized view refresh
- **Validation suite**: object audit and row parity checks
- **Docker support** for containerized execution
- **PyInstaller binary** for standalone distribution
- **CI/CD pipeline** with flake8 linting, Docker build, and artifact publishing

## Compatibility

- PostgreSQL 10+
- Python 3.10+
